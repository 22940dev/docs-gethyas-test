A basic Hyas project would be structured like this:

```
my-hyas-site/
├── archetypes/
├── assets/
├── config/
├── content/
├── data/
├── functions/
├── layouts/
├── node_modules/
├── public/
├── resources/
├── static/
├── .eslintrc.json
├── .stylelintrc.json
├── netlify.toml
├── package.json
└── yarn.lock
```
