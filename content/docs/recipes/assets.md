## Add a Lambda function

See also: [Functions]({{< ref "netlify#functions" >}})

### Example

```bash
./assets/lambda/hi-from-lambda.js
```

```js
exports.handler = (event, context, callback) => {
  callback (null, {
    statusCode: 200,
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      message: 'Hi from Lambda.',
    }),
  });
}
```
